Creative Coding

I've been using the open source Processing software recently to for creative exploration via code.

  • Damped Pendulum Clock: This is a digital prototype for a physical clock that I plan to build. The idea is to show the never-stopping quality of time using pendulums and a damping/forcing function that swings periodically about the current time. The physical clock will use small hobby servos and a microcontroller to simulate the pendulum motion.

    GITHUB LINK

  • Wisps: I love how organic these quickly become...amazing what a few sine/cosine functions and several octaves of perlin noise can create!

    GITHUB LINK

  • Noise: Similar in philosophy to the Wisps code above.

    GITHUB LINK

  • Random vs. Mitchell's Best Candidate: On the Left, a dot is generated at the coordinates: random(0,width), random(0,height). On the Right, Mitchell's Best Candidate algorithm is used to generate the dot's location. Mitchell's generates a pseudo-random distribution that is much more evenly distributed than a truly random distribution.

    GITHUB LINK LEFT and GITHUB LINK RIGHT

  • Rings: Simple exploration of classes and instances in Processing. Each click produces a new ring instance which continues to grow and interact with other instances until it reaches the edge of the screen and then disappears.

    GITHUB LINK

  • 03/14/15 Pi-Day: My first real Processing sketch, and arguably one of my more complicated. The first 10,000 digits of Pi are displayed sequentially on the screen. As the numbers representin the ASCII character codes for the letters of my company's motto appear, they're highlighted and the slogan is built-up.
