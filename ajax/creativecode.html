Creative Coding

I've been using the open source Processing software recently to explore creative code.

  • Damped/Forced Pendulum Clock Prototype: This is a digital prototype for a physical clock that I want to build when I find some free time. The idea was to show that time never stops using pendulums, as in a grandfather clock, and a damping/forcing function that undulates about the current minutes. The clock will use servo motors and a microcontroller to generate the motion.

    GITHUB LINK

  • Perlin Noise Wisps: I love how organic these become...amazing what a few sine/cosine functions and several octaves of perlin noise can create.

    GITHUB LINK

  • Turbulent Noise: Similar to the Perlin Noise above, this is about 16 lines of code to generate these intricate images.

    GITHUB LINK

  • Random vs. Mitchell's Best Candidate: On the LHS, a dot is generated at the coordinates (random(0,width), random(0,height)). On the RHS, the Mitchell's Best Candidate algorithm is used to generate the dot's location. Mitchell's generates a pseudo-random distribution that appears much more evenly distributed than a truly random function.

    GITHUB LINK LHS and GITHUB LINK RHS

  • Raindrops: A simple exploration of classes in Processing. Each click produces a new instance which continues to grow and interact with any other instance until it reaches the edge of the screen and disappears.

    GITHUB LINK

  • 03/14/15 Pi Day at work: My first real Processing sketch, and arguably one of the most complicated. The first 10,000 digits of Pi are displayed sequentially on the screen. As the ASCII character codes for my company's slogan appear in the code, they're highlighted.

    GITHUB LINK
