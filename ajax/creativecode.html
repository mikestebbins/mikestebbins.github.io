Creative Coding

I've been using the open source Processing software recently for creative exploration via code.

  • 03/14/15 (Pi-Day): My first real Processing sketch, and arguably one of my most complicated. The first 10,000 digits of Pi are displayed sequentially on the screen as the digit count is tracked in the LH corner. As the digits representing the ASCII character codes for the letters of my company's motto appear ("DESIGN THIS DAY"), they're highlighted and the motto gradually builds up in the bottom-center.

    GITHUB LINK

  • Damped Pendulum Clock: This is a digital prototype for a physical clock that I want to build sometime. The idea is to show the constant motion of time using pendulums and a damping/forcing function that keeps them swinging about the current time. The physical clock will use small servos and a microcontroller to simulate the damped periodic motions.

  • Pick It: This was inspired the Discovery Toy's game Pick It. It's a simple (but challenging) rule set.

    GITHUB LINK

  • Entanglement: This was a short-sprint effort inspired by this NPR Invisiblia podcast on the idea of Entanglement, from the atomic level to social levels.

    GITHUB LINK

  • Wisps: I love how organic these quickly become...amazing what a few sine/cosine functions and several octaves of perlin noise can create!

    GITHUB LINK

  • Noise: Similar in philosophy to the Wisps code above.

    GITHUB LINK

  • Random vs. Mitchell's Best Candidate: On the Left, a dot is generated at the coordinates: random(0,width), random(0,height). On the Right, Mitchell's Best Candidate algorithm is used to generate the dot's coordinates. Mitchell's generates a pseudo-random distribution that is more evenly distributed than a truly random distribution.

    GITHUB LINK LEFT and GITHUB LINK RIGHT

  • Rings: Simple exploration of classes and instances in Processing. Each click produces a new ring that continues to grow until it reaches the edge of the screen, where it disappears.
