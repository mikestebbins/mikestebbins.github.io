Homemade 3D Printer

In the spring of 2013, I finished a project that had been on and off-the-burner for about 2 years; a RepRap FDM 3D printer built from scratch. The printer features Clonedel (cast-urethane plastic) junction pieces, hand-cut frame and linear slide rods, linear bearings, a Bowden extruder, a J-Head hot-end, and an ABS/PLA-capable heated build-plate. It is controlled wirelessly via a Raspberry Pi single-board computer running the OctoPi print server, allowing me to kick off and monitor prints from any room in the house!
